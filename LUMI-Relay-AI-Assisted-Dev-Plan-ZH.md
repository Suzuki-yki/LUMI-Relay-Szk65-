# Week 2｜AI 辅助开发计划

## 我要做的最小功能是什么？

**LUMI Relay** 让用户修改基础 LUMI 的一个元素——发型、服装或配色——生成 Remix。用户连接钱包后，可将它作为 NFT 发布到 Monad 测试网，并记录它的父角色。

## 谁会使用它？

不懂区块链、但喜欢换装游戏、头像创作和分享的人；也包括 Monad Builder Camp 参与者。

## 用户完成的一个动作是什么？

用户点击 **“接力这只 LUMI”**，修改一个元素，再点击 **“发布我的 Remix”**。

## 我需要读哪 1–3 个文档？

1. Monad 测试网网络信息文档；
2. Monad Remix 部署合约指南；
3. OpenZeppelin ERC-721 文档。

## 本周真实实现什么？哪些可以 mock？

### 本周真实实现

- 可选择素材的 LUMI 角色预览页面；
- 钱包连接和最小 ERC-721 合约；
- `createRemix`：记录父角色 ID 和素材组合；
- 部署到 Monad 测试网，并发布一只 Remix NFT。

### 本周可以 mock

- 固定的 2D 角色素材；
- 社区画廊的本地假数据；
- 自定义素材上传、AI 审核、Marketplace、投票和奖励系统。

## 我如何证明它做出来了？

我会提交 GitHub Repo、前端截图或录屏、Monad 合约地址、Remix 交易哈希，以及一段 **“选择 → 连接 → 发布”** 的短视频。
